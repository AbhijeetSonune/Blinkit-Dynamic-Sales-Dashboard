Blinkit Dynamic Sales Dashboard

The Blinkit Dynamic Sales Dashboard is a Power BI project developed to provide a detailed and interactive analysis of Blinkit's sales data. It helps in tracking important business KPIs and visualizing trends across different outlet types, item categories, and locations.

Objective
The objective of this dashboard is to convert raw sales data into meaningful business insights that can help in monitoring performance, identifying trends, and supporting data-driven decisions.

Key Features
Interactive dashboard built in Power BI
Tracks Total Sales, Average Sales, Number of Items, and Average Rating.
Analyzes sales by Outlet Establishment
Compares performance by Outlet Size and Outlet Location.
Visualizes sales contribution by Item Type
Shows breakdown by Fat Content and Outlet Type.
Includes dynamic filters for better exploration and analysis.

Tools Used
Power BI
Excel / CSV Dataset
Data Cleaning and Transformation
Data Visualization

Benefits
Easy monitoring of sales performance.
Better understanding of outlet and product trends.
Supports faster and smarter business decisions.
Provides clear and interactive visual insights.

Conclusion
This dashboard is a useful business intelligence solution for analyzing Blinkit sales data in an effective and visually appealing way. It helps users explore key metrics, compare categories, and gain valuable insights from the data.
